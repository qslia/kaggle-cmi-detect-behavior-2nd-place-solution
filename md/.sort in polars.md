## What is `.sort()` in Polars?

```python

```

It means:

> **Order the rows in a predictable, alphabetical (or numeric) way
> using these columns, from left to right.**

Think of it like sorting a spreadsheet.

---

## How it works (step by step)

Polars sorts **lexicographically**:

1. First by `orientation`
2. If `orientation` is the same â†’ by `gesture`
3. If both are the same â†’ by `behavior`

### Example (before sort)

| orientation | gesture | behavior |
| ----------- | ------- | -------- |
| left        | swipe   | fast     |
